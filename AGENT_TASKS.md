# Tareas activas para agentes

## Prioridad alta
- Mantener la UI sin ejemplos falsos cuando el usuario pida paneles de produccion.
- Verificar cada cambio visual con captura o build antes de marcarlo terminado.
- Mantener la integracion MuAPI protegida en servidor, nunca en frontend.

## Prioridad media
- Normalizar textos largos para que no rompan paneles.
- Documentar modelos y requisitos obligatorios por IA antes de enviar payloads.
- Revisar botones de descarga, vista previa, restauracion y papelera por panel.

## Reglas operativas
- No ejecutar deploy.
- No hacer push a GitHub.
- No instalar stacks alternativos dentro del proyecto sin pedido concreto.
